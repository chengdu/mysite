MetaLogic
Version 1.2

The productive util used to organise programming assets.

MetaLogic is a small util developed to effectively manage programming assets, include shell snippets, code blocks, sample code files, libraries, packages, Web links and other resources. It's also a powerful util used to classify and organise programming resource by topics, categories, functions, programming languages, key words etc. MetaLogic help people organise large amount of programming resources in a very flexible way, and has complete freedom to define all kinds of metadata fields for management of classification and organisation.





If you can't open the link above, please launch App Store and found MetaLogic by searching.

Type of Programming Assets

  • Local file and folder.
  • Shell snippets.
  • Configuration files.
  • Code blocks.
  • Sample files.
  • Programming notes.
  • Web links.
  • Scripts. Problem resolutions, valuable scripts for reuse, compile, test, distribution etc.
  • Other type of resource.

Features

  • Built-in full featured code editor.
  • Support almost all common programming language syntax highlighting.
  • Tree style meta fields organisation and management. Fast, intuitive and simple.
  • Organise arbitrary contains relation of meta fields by drag and drop operation in meta fields tree.
  • Powerful database query by processing arbitrary combined logical operations.
  • Precise logical operation on any meta field, and their arbitrary combined logical operations.
  • Organise thousands of files by metadata, such as topic, category, function, self-defined tags etc.
  • Unlimited self-defined meta fields management.
  • Persisted bookmarks to fast locate lines in code blocks.
  • Favourites used to reduce query scope.
  • Database can be switched for exchange purpose or storage extension etc.
  • Support self-defined colour scheme.
  • Export report of the queried results from database.
  • Ultra lightweight.

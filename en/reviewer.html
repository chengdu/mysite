Chinese Chess Reviewer

The powerful Chinese chess machine

Chinese Chess Reviewer (aka CCReviewer), is a powerful chess machine. It was designed to record chess moves in international competitions. It have recorded thousands of large scale international chess games and you can use it to analyse these records and even participate in any of them at anytime.

This is the version of CCReviewer for iOS.
CCReviewer is a carefully designed chess review and learning tool on mobile device. It is a powerful chess analyser, move recorder and game records player. You can review game records and see the entire game process at any detailed steps. Learn how the masters playing Chinese chess in real games from 1980s. It's also an app to learn how to play Chinese Chess in an efficient way. The app has a build-in database that collected more than 40,000 games by 4000 professional players, so much master's moves that you want to deeply learning included!

Features

  • Autoplay or manually playback. You can playback any game in the built-in huge database step by step. You can also review the game records, see chess move automatically step by step, learn how chess stratagem applied in real games visually.
  • Unlimited undo & redo. You can cancel moves or redo moves at any time, no matter in autoplay mode or real game mode. That is, forward or backward any move steps just like dragging the progress bar in your music or video player.
  • Input game data. You can input game data and then analyse them later. Batch adding records also supported.
  • Record moves. The app can record each of your move and reload and playback them at any time.
  • Alternative stratagems. The app even allow you go to or back to any move, insert your own moves and add annotation to any single move step, and all of changes will be automatically saved. Try different stratagems on same game is a very important approach to analyse chess game. This is a very important feature that give your the ability to participate in the history games or any new added games, and even edit any of the records. You can travel with the time machine back to that moment and game with masters at any time point. What an exciting thing it is!
  • Analysis. Analyse games from the build-in huge database or your new added games by combine different search conditions.
  • Real game mode. Support full screen real game mode with time counter.
  • Write comments. Add or edit your note at any game or on any single move step, arbitrarily!
  • Bookmark. Mark your favourite or frequently used and review them in a convenient way.
  • Remember. The app can remember your last search items, last added or opened items etc, so intimate design.

Input Game Records

Step 1. Check "Full Screen" on and then click "Play".
Step 2. Click "Home" icon on the left corner of the bottom, and then select "Save moves" when the app prompted a dialog box, here, you have two options:
   a. Directly input event name and game data when add your first game record. The app will automatically create a new Event for you and add your data as a new record to it.
   b. Select an event and then add record to it after you have already created at least one event.
Step 3. Click "Cancel" on the "Select an event" window then the app will allow you to create a new "Event" just like "option a", input game data then click "Save".

Other Situations
a. You can play game records from a search, add some new moves and then update the game records.
b. You can directly play in "Full Screen" mode and then save your moves.

Tips

  • In AI mode, black first by default.
  • In real game mode, red first by default.
  • Move finger. The chess rules will be applied automatically when you move your finger, the chess will back to its start position if target is illegal.
  • You can scroll the narrow area near the bottom to see more info when autoplaying searched records.
